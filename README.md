# KUEnyang 🍰

**KUEnyang** adalah aplikasi berbasis digital yang dikembangkan untuk memudahkan manajemen operasional toko kue.

## 🚀 Fitur Utama
* **Manajemen Data:** Pengelolaan informasi secara terstruktur.
* **User Interface:** Desain antarmuka yang intuitif dan responsif.

## 🛠️ Stack Teknologi
* **Frontend Backend:** Visual Studio sebagai IDE utama.
* **Version Control:** Git & GitHub.
* **Development Environment:** .NET / C#.
